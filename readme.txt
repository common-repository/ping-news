== Description ==

The Ping! WordPress plugin allows hyperlocal journalists to submit their news stories to the Ping! system and receive approval/feedback on their stories.

== Installation ==

Upload the Ping! plugin to your blog, activate it, then enter your authorization token taken from the Ping! system.
